# ThermoSpec 🛠️

---

**ThermoSpec** — это веб-приложение для интеграции, нормализации и поиска данных из разнородных офлайн-каталогов инженерного оборудования.

Проект разработан для автоматизации работы инженеров в проектных бюро. Система позволяет уйти от хаотичного поиска по сотням разрозненных файлов поставщиков, собирая данные в единую локальную базу с возможностью гибкого поиска, сравнения и экспорта.

## 🎯 Какую задачу решает проект?

В инженерной практике каталоги оборудования часто поставляются не только в виде онлайн каталогов, но и в офлайн форматах (PDF, XLSX, DOCX) и с абсолютно разной структурой характеристик. **ThermoSpec** автоматизирует этот хаос:

1. **Умный импорт:** позволяет парсить данные из файлов, используя классические алгоритмы и ИИ, а затем применять к ним маппинг и трансформации
2. **Гибкая структура данных:** использует модель **EAV (Entity-Attribute-Value)**, что позволяет без перепроектирования БД добавлять новые категории оборудования и любые кастомные атрибуты к ним
3. **Рабочее место инженера:** позволяет искать оборудование по локальной базе, сравнивать модели между собой, группировать их в проекты и выгружать результаты в Excel для дальнейшей работы

---

## 💻 Функционал приложения

- **Импорт данных:** загрузка файлов (PDF, Word, Excel), извлечение технических характеристик с помощью классических алгоритмов и LLM, нормализация данных
- **Поиск и фильтрация:** полнотекстовый поиск, динамическая фильтрация по атрибутам выбранной категории оборудования
- **Панель сравнения:** сравнение характеристик нескольких моделей оборудования с подсветкой различий
- **Управление проектами:** возможность создавать проекты и добавлять туда выбранные позиции оборудования
- **Экспорт:** выгрузка сформированных списков оборудования из проектов в таблицы Excel для дальнейшей работы
- **Управление структурой каталога:** управление категориями и их атрибутами: настройка специфичных параметров (например, мощность, давление, диаметр) для каждой группы оборудования

## 🛠️ Стек технологий

### Frontend

- **Ядро:** React 19, TypeScript, Vite
- **Интерфейс:** TailwindCSS v4, HeroUI, Lucide React, @dnd-kit
- **Управление состоянием:** Zustand, TanStack Query

### Backend

- **Ядро:** Node.js (Express 5), TypeScript
- **Хранение и управление данными:** PostgreSQL, Prisma ORM, AWS S3 SDK
- **Интеграция ИИ:** OpenAI SDK

## ⚙️ Быстрый старт

### Требования

Убедитесь, что у вас установлена **Node.js версии 20.x или выше** и запущен инстанс PostgreSQL

### Установка

1. Клонируйте репозиторий:

   ```bash
   git clone https://github.com/joat21/ThermoSpec.git
   cd ThermoSpec
   
   ```
2. Установите зависимости для всего монорепозитория:

   ```bash
   npm install
   
   ```
3. Настройте переменные окружения (создайте файлы .env в /backend и /frontend на основе примеров .env.example)

### Настройка базы данных

1. Убедитесь, что в /backend/.env указана корректная строка подключения к PostgreSQL
2. Примените схему для создания структуры таблиц:

   ```bash
   npx prisma db push --schema=backend/prisma/schema.prisma
   
   ```

### Запуск в режиме разработки

Запуск фронтенда и бэкенда одной командой:

```bash
npm run dev
```

- Фронтенд будет доступен на: http://localhost:5173
- Бэкенд будет доступен на: http://localhost:8080
